# 🍔 Cardápio Digital

Um projeto didático de cardápio digital desenvolvido para explorar conceitos fundamentais de **HTML** e **CSS** no contexto de um Fullstack.

## 📚 Contexto do Projeto

Este projeto foi criado como atividade prática após o módulo de HTML e CSS de um curso de Desenvolvimento Fullstack, com o objetivo de consolidar o aprendizado em:
- Semântica HTML
- Estilização com CSS
- Estruturas de layout responsivas
- Organização de componentes

---

## 🛠️ Tecnologias Utilizadas

### **HTML5**
- **Impacto**: Forneceu a estrutura semântica e acessível para o cardápio, permitindo que navegadores e ferramentas assistivas entendam o conteúdo de forma apropriada
- **Decisões-chave**:
  - Uso de `<header>` e `<nav>` para melhor organização semântica
  - Links internos com `#ids` para navegação entre seções (Hamburgueres, Acompanhamentos, Bebidas)
  - Estrutura clara de divisão de produtos com `<div class="boxProduto">`

### **CSS3**
- **Impacto**: Transformou a estrutura HTML em uma experiência visual atraente e profissional, com foco em usabilidade
- **Decisões-chave**:
  - **Flexbox**: Utilizado para layouts responsivos (`display: flex`), permitindo alinhamento perfeito dos elementos sem necessidade de frameworks complexos
  - **Cores corporativas**: Paleta com laranja (#ffa300) e cinza escuro (#1e1f22) para identidade visual clara e contraste legível
  - **Transparência RGBA**: Aplicada aos boxes de produtos para criar profundidade e mantém a visibilidade do fundo
  - **Reset CSS**: `* { margin: 0; padding: 0; box-sizing: border-box; }` garante consistência entre navegadores

### **Imagens**
- **Impacto**: Enriquecimento visual do cardápio, tornando mais apetitoso e profissional
- **Estrutura**: Pasta `/imagens` organizada com logo, produtos e background
